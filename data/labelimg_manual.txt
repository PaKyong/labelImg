labelimg 1.8.7.1 / Manual

File)
이전 라벨 파일 불러오기 (Ctrl + G)
- 직전 이미지의 라벨 파일 정보를 현재 라벨에 추가한다.
- 자동 저장은 되지 않으며 유저가 편집한 이후 저장 가능하다.
- 이전 라벨 파일의 라벨 -> 현재 라벨의 라벨 순서대로 정리되어 로드된다.


Mode)
방향키 사용시 모드 변경 (Z/X/C)
- 모든 입력은 Shift + 방향키 = 10픽셀, Ctrl + 방향키 = 100픽셀이 적용된다.
- 편집모드에서만 사용 가능.

이동 모드 (Z)
- 방향키 입력시 선택한 라벨 박스가 1픽셀씩 이동한다.

확장 모드 (X)
- 방향키 입력시 선택한 라벨 박스가 해당 방향으로 1픽셀씩 넓어진다.

축소 모드 (C)
- 방향키 입력시 선택한 라벨 박스가 해당 방향으로 1픽셀씩 좁아진다.

라벨 박스 프리셋 저장 (Ctrl + 1~5)
- 원하는 라벨 박스를 선택한 이후 Ctrl + 1~5를 눌러 저장 할 수 있다.
- 우측의 프리셋 텍스트박스를 이용해 저장된 프리셋의 이름을 확인 할 수 있다.

프리셋 사용 (1~5)
- 1~5번 키를 눌러서 저장해놓은 라벨 박스를 바로 입력 할 수 있다.
- 이미지 크기보다 밖에 있는 프리셋 라벨은 입력이 되지 않는다. 
(ex. 이미지 크기 800x600일 경우 900,700 좌표에 있는 프리셋은 사용되지 않는다.)

파일 이름 로그 저장 (M)
- 현재 파일명을 이미지 폴더/Savelog라는 폴더를 만들어 txt 파일에 저장한다.
- 파일명 마다 저장 이유를 물어보며 이유를 적을 수도, 적지 않을 수도 있다. (빈칸으로 입력)
- 이미 저장된 파일명은 다시 저장되지 않는다.
- Cancel 선택시 파일 이름은 저장되지 않는다.
- 유저가 직접 txt 파일을 수정해도 동작에는 문제가 없다.

파일 이름 로그 불러오기 (Shift + M)
- 저장되어있는 로그 파일을 불러온다.
- 현재 날짜의 파일을 불러오며 존재하지 않지만 이전 날짜의 로그 파일이 존재할시 탐색기를 오픈한다.
- 아무 파일도 존재하지 않는 경우 파일을 불러오지 않는다.

이미지 영구 삭제 (Shift + Delete)
- 삭제하고 싶은 이미지가 나왔을때 눌러서 이미지를 삭제 할 수 있다.
- 이미지와 라벨 파일이 삭제되며, 삭제된 파일 이름이 /DeleteLog/라는 폴더에 텍스트 파일로 저장된다.
- 이미지가 1장만 남아있을 경우 삭제한 다음 파일 열기 창이 열린다.

이전 저장 파일 복원 (F8)
- 자동 복사 모드 사용시 백업 파일이 별도의 폴더에 생성된다.
- 해당 기능을 사용 할 시 백업 파일이 있다면 해당 파일에 저장된 상태로 되돌아간다.
- 만약 백업 파일이 없을 경우 복원은 불가능하며, 백업 파일은 이동할때마다 계속 새롭게 덮어씌워진다.


Shape)
실행 취소 / 다시 실행 (Ctrl + Z / Ctrl + Y)
- 별도의 undo / redo 윈도우에 작업 내역이 기록된다
- 실행 취소를 1회 할때마다 이전 작업으로 돌아가며, 다시 실행할경우 이후 작업으로 진행된다.
- 실행 취소를 진행한 상태에서 추가 작업을 할경우 다시 실행 내역은 지워진다.

이전 박스 선택 (Q) / 다음 박스 선택 (E)
- Q와 E를 눌러서 오른쪽 라벨 리스트의 순서대로 이전/다음 라벨을 바로 선택 가능하다.

라벨 박스 회전 (R)
- 라벨 박스를 선택 한 이후 R을 눌러서 라벨 박스를 회전 할 수 있다. (가로-세로를 바꿈)
- 회전시 라벨이 이미지 영역 밖으로 나가게 되는경우 회전이 되지 않는다.

라벨 박스 크기 수정 (B)
- 라벨 박스를 선택 한 이후 B를 누르면 입력 대기 상태로 바뀌며 오른쪽 텍스트 박스에 활성화 상태가 반영된다.
- 아무것도 클릭하지 않고 B를 한 번 더 누를경우 라벨 박스가 오른쪽 아래 방향으로 2배 크기로 커진다.(단 라벨이 커졌을때 화면 밖으로 나가는 경우 커지지 않는다) 
- 화면의 특정 지점을 클릭하고 B를 한 번 더 누를경우 라벨 왼쪽 위의 점과 클릭한 지점을 채우는 라벨로 모양과 크기가 변경된다.
- 크기 변경 모드를 활성화 할 경우 취소를 원한다면 F12를 눌러서 활성화 모드를 취소 할 수 있다.

선택된 라벨 자동 추적 (T)
- 라벨 박스를 선택 한 이후 T를 누르면 해당 라벨 박스가 마우스 커서 위치로 자동 추적된다.
- 다른 라벨을 선택하거나 선택 해제 할 경우, 또는 T를 한 번 더 누를 경우 추적이 해제된다.

선택된 라벨 확대 (Shift + Z)
- 라벨을 선택한 이후 입력하면 해당 라벨을 중심으로 이미지가 확대된다.

가로방향 자르기(F) / 세로방향 자르기(V)
- 라벨 박스를 선택 한 이후 F/V를 누르면 입력 대기 상태로 바뀌며 오른쪽 텍스트 박스에 활성화 상태가 반영된다.
- 아무것도 클릭하지 않고 F/V를 한 번 더 누를경우 라벨 박스가 2등분되며, 라벨 박스의 특정 지점을 클릭하고 F/V를 누를경우 해당 클릭 지점에서 분할된다.
- 자르기 모드를 활성화 할 경우 취소하고 싶다면 F12를 눌러서 취소가 가능하다.

십자 자르기 (G)
- 십자로 잘라서 4개의 라벨을 만드는 기능
- 사용법은 가로/세로 자르기와 동일하다.

라벨 합치기 (H)
- 2개 이상의 라벨을 선택한 이후 라벨을 합칠 수 있다.
- 라벨은 모든 라벨의 최대 영역으로 합쳐진다. (가장 작은값 / 가장 큰값)
- 라벨명이 모두 같을 경우 별도로 물어보지 않으며, 라벨명이 다를경우 라벨명 입력창이 팝업된다.
- difficult 옵션값이 1개의 라벨에라도 입력되어 있을경우 자동으로 difficult 옵션이 입력된다.

자르기 활성화 취소 (F12)
- 자르기나 합치기 대기모드를 취소한다.

선택한 라벨 보이기/숨기기 (U)
- 라벨을 선택한 이후 U를 누를경우 숨겨지며, 오른쪽 라벨 리스트에서 숨겨진 라벨을 선택후 U를 누르면 보여진다.
- 라벨 리스트의 체크박스와 기능이 동일하다.


Help)
labelimg 설명서 (F1)
- labelimg의 설명서를 연다.


Right Tool Bar)
이전 라벨 파일 자동 복사 모드 (Ctrl + Shift + M)
- 이전 이미지의 라벨 파일을 자동으로 복사해서 현재 라벨 파일과 합쳐서 저장 한 후 불러온다.
- 단 라벨명과 라벨 크기, 위치가 정확히 동일한 라벨은 중복으로 분류되어 걸러진다.
- 순서는 이전 라벨 -> 현재 라벨 순서대로 정리되어 로드된다.

특정 라벨 자동 추가 모드 (F9)
- 라벨을 저장한 이후(Ctrl + F9) 모드를 선택해서 자동으로 라벨을 입력 할 수 있다.
- 라벨이 중복되는 경우 입력되지 않는다.
- 저장된 라벨이 없거나 이미지 범위를 벗어날 경우 에러 메시지가 출력된다.

박스 수정 잠금 모드 (K) / 박스 이동 잠금 모드 (L)
- 박스의 크기수정을 잠그거나, 이동을 잠글 수 있다. 
- 두 기능을 모두 켤 경우 라벨의 마우스 편집은 불가능해진다.
- 해당 잠금 모드는 마우스 사용에 한하며, 키보드 수정이나 자동 추적 모드의 경우는 잠그는 것이 불가능하다.


Label Select)
라벨 여러개 동시 선택 (Shift + 클릭 or 드래그)
- 편집 모드에서 Shift를 누른 상태에서 라벨을 클릭하거나 드래그 하면 라벨을 여러개 동시 선택 할 수 있다.
- 드래그로 선택시 라벨은 추가만 가능하며, 클릭으로 선택 할 경우는 선택된 라벨을 한 번 더 클릭할시 선택이 해제된다.
- 동시 선택된 라벨들은 녹색으로 표시되며, 1개의 라벨만 선택된 경우 기존과 동일하게 X마크로 표시된다.
- 여러개를 동시 선택후 라벨 삭제(Delete), 라벨 이름 수정(Ctrl+E), 프리셋 저장(Ctrl+1~5, Ctrl+F9), 라벨 이동, 라벨 숨기기(U)가 가능하다.
- 복수의 라벨을 프리셋 저장할 경우 확인창에는 저장된 프리셋의 개수가 표시된다. (ex. 2 Shapes, 3 Shapes, 6 Shapes)

라벨 여러개 동시 이동 (여러 라벨 선택한 이후 -> Ctrl + 드래그)
- 라벨을 여러개 동시 선택한 상태에서 Ctrl을 누른채로 라벨을 드래그 할 경우 동시 이동이 가능하다.
